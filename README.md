# React + antd + Webpack StartKit

StartKit 项目

## Feature
- 支持多页面类型的React应用开发
- 使用 webpack-dev-server
- 支持 React 热加载
- 支持 CSS 及 JS 的热加载

## 使用的技术

- [x] [Webpack](https://webpack.github.io)
- [x] [React](https://facebook.github.io/react/)
- [x] [Babel](https://babeljs.io/)
- [x] [antd](https://ant.design/)

## 使用方法

```
**1. 安装依赖
npm install
**2. 启动
npm start
